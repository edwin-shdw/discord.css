# Guildbar

Don't want to list servers with a basic list? Use the `dc-guildbar` class to create guildbar containing DMs and guilds. To
display an item as active add the `dc-active` class to the `dc-guildbar-item` class. If you want to separate items you
can use the `dc-guildbar-separator` class on a `<hr>`.

</CodePreview>

```html

    <div class="dc-guildbar-item dc-active">
        <img class="dc-guildbar-item-img" src="/img/profile/shabotto.png">
```

## Guilds without image

Guilds which don't have any image set will display the first letters of the guild name in uppercase e.g. `My server`
will be displayed as `MS`. Furthermore, the font-size will shrink depending on the length of the initials. It's not
possible to cover this with pure CSS so this will require a JS solution. You can use the `dc-guildbar-item-letter` inside
a container with the `dc-guildbar-item-img` class to style the letters and icons.

You can also use Icons instead of letters. The icons will be displayed in the same way as the letters. Simply add a
`<svg>` element inside the `dc-guildbar-item-img` class and apply the `dc-guildbar-item-letter` class to it.

<CodePreview>
  <GuildbarLetterExample />
</CodePreview>

```html
<div class="dc-guildbar dc-scrollbar-none" style="max-block-size: 16rem;">
  <div class="dc-guildbar-item">
    <div class="dc-guildbar-item-img">
      <span class="dc-guildbar-item-letter">MS</span>
    </div>
  </div>
  <div class="dc-guildbar-item">
    <div class="dc-guildbar-item-img">
      <span class="dc-guildbar-item-letter">MSI</span>
    </div>
  </div>
  <div class="dc-guildbar-item">
    <div class="dc-guildbar-item-img">
      <span class="dc-guildbar-item-letter" style="font-size: 16px;">MSIT</span>
    </div>
  </div>
  <div class="dc-guildbar-item">
    <div class="dc-guildbar-item-img">
      <span class="dc-guildbar-item-letter" style="font-size: 14px;">MSITL</span>
    </div>
  </div>
  <div class="dc-guildbar-item">
    <div class="dc-guildbar-item-img">
      <svg class="dc-guildbar-item-letter" xmlns="http://www.w3.org/2000/svg" width="20" height="20" viewBox="0 0 24 24">
        <circle cx="12" cy="12" r="10" fill="transparent"></circle>
        <path fill="currentColor" fill-rule="evenodd" d="M12 23a11 11 0 1 0 0-22 11 11 0 0 0 0 22Zm0-17a1 1 0 0 1 1 1v4h4a1 1 0 1 1 0 2h-4v4a1 1 0 1 1-2 0v-4H7a1 1 0 1 1 0-2h4V7a1 1 0 0 1 1-1Z" clip-rule="evenodd"></path>
      </svg>
    </div>
  </div>
</div>
```

:::warning Deprecation
The success variant of the guildbar is no longer actively used in the newer Discord UI. Nevertheless, it will remain in
the library for the time being. If you want to use it, you can do so with the `dc-guildbar-item-success` class.
:::
