# Sidebar

Don't want to list servers with a basic list? Use the `dc-sidebar` class to create sidebar containing DMs and guilds. To
display an item as active add the `dc-active` class to the `dc-sidebar-item` class. If you want to separate items you
can use the `dc-sidebar-separator` class on a `<hr>`.

    <SidebarExample />
</CodePreview>
