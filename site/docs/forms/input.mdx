# Input

Input fields require the `dc-input` class while text. Labels should get the `dc-label-header` class while the more
detailed description of an input should get the `dc-label-description` class.

<CodePreview>
  <div>
    <Label type='header'>A label header</Label>
    <Label type='description'>Labels can be described by an additional text which will look like this.</Label>
    <Input type="text" placeholder="Simple inputfield" />
  </div>
</CodePreview>

```html
<div>
    <label class="dc-label-header">A label header</label>
    <p class="dc-label-description">
        Labels can be described by an additional text which will look like this.
    </p>
    <input class="dc-input" type="text" placeholder="Some input">
</div>
```


Textareas will need the `dc-textarea` class.

<CodePreview>
    <Label type='header'>A label header</Label>
    <Label type='description'>Labels can be described by an additional text which will look like this.</Label>
    <Textarea type="text" placeholder="A textfield" rows={9} />
</CodePreview>

```html
<label class="dc-label-header">A label header</label>
<p class="dc-label-description">
    Labels can be described by an additional text which will look like this.
</p>
<textarea class="dc-textarea" placeholder="Some input">
```
