# Radio

Radios can be displayed with the `dc-radio` class.

<CodePreview bordered>
  <Radio id="radio1" name="radioExample">Radio 1</Radio>
  <Radio id="radio2" name="radioExample">Radio 2</Radio>
  <Radio id="radio3" name="radioExample" defaultChecked>Radio 3</Radio>
</CodePreview>

```html
<label class="dc-radio" for="radio1">
    <input type="radio" id="radio1" name="radioExample">
    <span>Radio 1</span>
</label>
<label class="dc-radio" for="radio2">
    <input type="radio" id="radio2" name="radioExample">
    <span>Radio 2</span>
</label>
<label class="dc-radio" for="radio3">
    <input type="radio" id="radio3" name="radioExample" checked>
    <span>Radio 3</span>
</label>
```

## Disabled

Add the disabled attribute to apply the grayed out style to both the `<input>` and `<label>` tag.

<CodePreview bordered>
  <Radio id="radioDisabled1" name="radioExample2">Radio 1</Radio>
  <Radio id="radioDisabled2" name="radioExample2">Radio 2</Radio>
  <Radio id="radioDisabled3" name="radioExample2" disabled>Radio 3</Radio>
</CodePreview>

```html
<label class="dc-radio" for="radioDisabled1">
    <input type="radio" id="radioDisabled1" name="radioExampleDisabled">
    <span>Radio 1</span>
</label>
<label class="dc-radio" for="radioDisabled2">
    <input type="radio" id="radioDisabled2" name="radioExampleDisabled">
    <span>Radio 2</span>
</label>
<label class="dc-radio" for="radioDisabled3">
    <input type="radio" id="radioDisabled3" name="radioExampleDisabled" disabled>
    <span>Radio 3</span>
</label>
```
