# Switch

You can build a switch by wrapping a `<input>` element with the `dc-switch` class and adding a `dc-switch-slider` class
as sibling.

<CodePreview>
  <div className="dc-switch-wrapper">
    <Switch/>
  </div>
</CodePreview>

```html
<div class="dc-switch">
  <input type="checkbox">
  <span class="dc-switch-slider"></span>
</div>
```

Discord usually displays the label and switch with a `space-between`. For this case you can use the `dc-switch-wrapper`
and `dc-switch-label` classes.

<CodePreview>
  <div className="dc-switch-wrapper">
    <label className="dc-switch-label" htmlFor="switch">Activate DCS for current project</label>
    <Switch id="switch"/>
  </div>
</CodePreview>

```html
<div class="dc-switch-wrapper">
    <label class="dc-switch-label" for="switch">Activate DCS for current project</label>
    <div class="dc-switch">
        <input type="checkbox" id="switch">
        <span class="dc-switch-slider"></span>
    </div>
</div>
```

:::info
Even if you don't need the label, you should wrap your switch in an element with the `dc-switch-wrapper` class.
Otherwise, the `:focus-visible` selector won't mark the switch as focused for keyboard users.
:::


## Disabled

Add the disabled boolean attribute on the `<input>` to give the switch a grayed out appearance, remove pointer events
and prevent focusing.

<CodePreview>
  <div className="dc-switch-wrapper">
    <label className="dc-switch-label" htmlFor="disabledSwitch">Activate DCS for current project</label>
    <Switch id="disabledSwitch" disabled />
  </div>
</CodePreview>

```html
<div class="dc-switch-wrapper">
    <label class="dc-switch-label" for="disabledSwitch">Activate DCS for current project</label>
    <div class="dc-switch">
        <input type="checkbox" id="disabledSwitch" disabled>
        <span class="dc-switch-slider"></span>
    </div>
</div>
```
